3.Different types of server ?
Web,Application,Data , File 
1️⃣ Web Server (HTTP,HTTS,Frontend,HTML,CSS,JS)



What it does: Handles HTTP/HTTPS requests
Serves: HTML, CSS, JS, images
Examples:
Apache
Nginx
IIS
Example use: Serves your React frontend


2️⃣ Application Server (business,Apis,Authentication,Node.js, MERN backend )



What it does: Runs business logic
Serves: APIs, authentication, processing
Examples:
Node.js (Express, Nest)
Java Spring Boot
Example use: Your Express backend in MERN

3️⃣ Database Server (store and manage,)

What it does: Stores and manages data
Examples:
MongoDB
MySQL
PostgreSQL
Example use: MongoDB storing users, properties, orders


4.File Server  (store and share,FTP,)

What it does: Stores and shares files
Examples:
FTP Server
AWS S3
Example use: Image uploads, documents





4.server vs database ?


1. What is a Server?(recive and send response)



✅ Definition
A server is a computer/system that receives requests and sends responses over a network.




✅ What it does (Handle,Logic,Access data,)



Handles client requests (browser, mobile app)
Runs application logic
Controls access to data
Communicates with databases

Client → Server → Response

2. What is a Database?
✅ Definition (system ,store , organizes,manage )

A database is a system that stores, organizes, and manages data.


✅ Examples
MongoDB
MySQL
PostgreSQL

Server → Database → Data

User (Browser)
   ↓
Server (Node + Express)
   ↓
Database (MongoDB)

Three-tier architecture is a software design pattern where an application is divided into three separate layers, each with a specific responsibility.

   ↓
Application Tier (Node + Express)
   ↓
